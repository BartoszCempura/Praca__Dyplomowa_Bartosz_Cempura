    Instalacja Flask i jego biblioitek
        pip install flask flask-jwt-extended flask-sqlalchemy Flask-CORS Flask-Mail pandas psycopg2-binary python-slugify Flask-APScheduler
    Instalujemy serwery developerskie i produkcuyjne:
        pip install gunicorn gevent
    Sprawdzamy poprawnośc instalacji za pomocą:
        pip list
    Nastepnie zapisujemy dane o zainstalowanych wtycznkach i ich wersji do pliku (w celu odtworzenia instalacji na innym urządzeniu)
        pip freeze > "../../Additional Files/requirements.txt" (uruchaiamy w folderze Praca__Dyplomowa_Bartosz_Cempura)

Tworze folder Frontend
    Instalacj React na scuffolding Vite (musze mieć wcześniej zainstalowane w systemie node.js)
        cd.. (wracamy do Webpage)
        npm create vite@latest frontend -- --template react
        cd frontend
        npm install
        npm install react-router-dom
    Instalacja Tailwind css wraz z Vite plugin oraz kolekcji klas daisyUI
        npm install tailwindcss@latest @tailwindcss/vite@latest daisyui@latest

    Następnie wykonujemy procedure konfiguracyjną zawartą pod tym adresem:
        https://daisyui.com/docs/install/react/
    Konfiguracja Vite + React + tailwind css umożliwa uruchomienie serwera dev za pomocą komendy:
    
        cd Webpage\frontend
        npm run dev
    
    Wszystkie dodatki i nowe klasy powinny teraz działąć. Podkreślenie @plugin "daisyui"; jest błędem Visual Studio Code

    instalujemy axios npm install axios

Usówamy niepotrzebne pliki z folderu Scripts

W pliku vite.config.js dodaje konfiguracje proxy aby nie pisać pełnej ścieżki włącznie z adresem serwera tylko sam adres do endpoint
 server: {
    proxy: {
      "/api": {
        target: "http://localhost:5000",
        changeOrigin: true,
        secure: false, 
      },
    },
  },
