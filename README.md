Praca inżynierska przedstawia pełen system e-commerce składający się z sklepu internetowego z panelem administracyjnym, mechanizmami statystycznymi i algorytmami rekomendacyjnymi. 
Backend został oparty o framework Flask z bazą danych PostgreSQL oraz ORM SQLAlchemy, natomiast frontend został zbudowany w technologii React. 
Zaimplementowano autentykacje JWT, koszyk zakupowy, finalizację transakcji, konfigurację konta użytkownika, śledzenie interakcji użytkowników, listę ulubionych, podpowiedzi podobnych produktów. 
Panel administratora oferuje wizualizacje danych pozyskanych za pośrednictwem algorytmów oraz zarządzanie transakcjami. 
Za pośrednictwem rozwiązania APScheduler wprowadzono automatyczne czyszczenie porzuconych koszyków zakupowych oraz aktualizacje wartości podobieństw produktów. 
Głównym zadaniem rozwiązania było uzyskanie wiedzy o działaniu i procesie produkcji systemu e-commerce co zostało w pełni spełnione
