Projekt stanowi Pracę Inżynierską zamykającą 3.5 letni okres studiów z dziedziny Informatyki na specjalizacji programowanie i Bazy danych. 
Jego przedmiotem jest pełen system e-commerce składający się z sklepu internetowego oraz panelu administracyjnego.

Zaimplementowano:
- autentykację JWT 
- koszyk zakupowy 
- finalizację transakcji
- konfigurację konta użytkownika
- śledzenie interakcji użytkowników
- listę ulubionych produktów 
- podpowiedzi podobnych produktów
- zarządzanie transakcjami
- automatyczne czyszczenie porzuconych koszyków
- automatyczna aktualizacja podobieństwa między produktami
- wizualizacje danych statystycznych

Backend został oparty o framework Flask z bazą danych PostgreSQL oraz ORM SQLAlchemy, natomiast frontend został zbudowany z wykorzystaniem technologii React. 
Automatyzacje działań zaimplementowano za pomocą rozwiązania APScheduler

Głównym celem pracy było uzyskanie wiedzy o działaniu i procesie produkcji systemu e-commerce, co w mojej ocenie zostało w pełni spełnione.

This project is an engineering thesis concluding a 3.5-year degree in Computer Science, with a specialization in programming and databases.
The project consists of a full e-commerce system, including an online store and an administrative panel.

Implemented features:
- JWT authentication
- shopping cart
- transaction processing
- user account management
- user interaction tracking
- wishlist functionality
- similar product recommendations
- transaction management
- automatic cleanup of abandoned carts
- automatic updates of product similarity data
- statistical data visualization

The backend is built using Flask with PostgreSQL and SQLAlchemy, while the frontend is developed with React. 
Task automation is implemented using APScheduler.

The main goal of the project was to gain practical knowledge of how e-commerce systems are designed and developed, which was successfully achieved.
